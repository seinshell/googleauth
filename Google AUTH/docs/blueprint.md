# **App Name**: Google Auth by DIV Corp.

## Core Features:

- Google Authentication: Implement Google sign-in/sign-up functionality using Firebase Authentication.
- User Profile Creation: Automatically create a user profile upon successful Google authentication.
- Secure Session Management: Maintain secure user sessions after successful login.
- Logout Functionality: Provide a secure logout option for users.
- UI Integration: Seamless integration of the authentication flow within the application's user interface.

## Style Guidelines:

- Primary color: Deep Blue (#1A237E) to convey trust and security.
- Background color: Light Grey (#F5F5F5) for a clean, modern look.
- Accent color: Electric Blue (#7DF9FF) to highlight interactive elements and calls to action.
- Font: 'Inter', a sans-serif for both body and headline to maintain a clean and accessible look. Note: currently only Google Fonts are supported.
- Use Google's official logo for the 'Sign in with Google' button.
- The Google Sign-In button should be prominently placed on the login screen.
- Use a subtle loading animation during the authentication process.
